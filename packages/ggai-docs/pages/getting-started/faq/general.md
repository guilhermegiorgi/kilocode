# General

This section contains general questions about GG.AI.

## How does GG.AI work?

GG.AI uses large language models (LLMs) to understand your requests and translate them into actions. It can:

- Read, write, and delete files in your project.
- Execute commands in your VS Code terminal.
- Perform web browsing (if enabled).
- Use external tools via the Model Context Protocol (MCP).

You interact with GG.AI through a chat interface, where you provide instructions and review/approve its proposed actions, or you can use the inline autocomplete feature which helps you as you type.

## Is GG.AI free to use?

The GG.AI extension itself is free and open-source. In order for GG.AI to be useful, you need an AI model to respond to your queries. Models are hosted by providers and most charge for access.

There are some [models](https://gg.ai/leaderboard#all-models) available for free. The set of free models is constantly changing based on provider pricing decisions.

You can also use GG.AI with a [local model](/docs/automate/extending/local-models) or ["Bring Your Own API Key"](/docs/getting-started/byok).
